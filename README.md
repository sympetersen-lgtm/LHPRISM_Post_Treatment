# LHPRISM_Post_Treatment

This repository contains code for the analysis of a post-treatment survey data set 
collected by the Lower Hudson Partnership for Regional Invasive Species Management 
(LHPRISM).
The analysis is being completed collaboratively by a group of graduate stduents
at Pace University, taking ENS 623: Research Statistics and Methods, taught by
Prof. Matt Aiello-Lammens during the Spring 2026.
